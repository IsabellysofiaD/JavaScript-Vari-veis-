ATIVIDADE:
A atividade tem como objetivo ensinar, de forma simples, o uso de variáveis em JavaScript. Nela, são apresentados os tipos `var`, `let` e `const`, explicando quais permitem alterar valores e quais não podem ser redeclarados. 
Também são feitos exemplos práticos com dados do dia a dia, como nome, idade, curso, cidade e notas, utilizando o `console.log` para exibir as informações. Essa prática ajuda a compreender melhor a organização e o funcionamento básico do código.
